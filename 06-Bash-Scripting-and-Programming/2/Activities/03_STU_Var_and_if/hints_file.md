# Using Variables
- To set a variable `x="SOME FILE/DIRECTORY PATH"`
- To use the above set variable use `$x`

# Using If Statements
- General if syntax:
```bash
if [ <condition> ]
then
  <run_this_command>
fi
```
- General if-else syntax:

```bash
if [ <condition> ]
then
  <run_this_command>
else
  <run_this_other_command>
fi
```
